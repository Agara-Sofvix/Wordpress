# Agara-Sofvix

Digital solutions for various industries.

## Project Structure
- `/admin`: React-based administration dashboard.
- `/backend`: Node.js/Express API server.
- `/frontend`: Main company website.

## Setup
1. Backend: `cd backend && npm install && npm start`
2. Admin: `cd admin && npm install && npm run dev`
3. Frontend: `cd frontend && npm install && npm run dev`

## SEO & Management
The project includes a comprehensive SEO management system and a secure admin login flow.
